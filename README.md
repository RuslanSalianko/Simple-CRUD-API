# Simple-CRUD-API

## Installation

```console
 $ npm install
```
or 

```console
 $ yarn install
```

## Run 
* dev
```console
$ npm run start:dev
```

```console
$ yarn start start:dev
```
* prod
```console
$ npm run start:prod
```

```console
$ yarn start start:prod
```
* multi (cluster)
```console
$ npm run start:multi
```

```console
$ yarn start start:multi
```
